# Elementor Pro’s Angie AI Agent – Complete Usage Guide

Elementor **Angie** is a context-aware AI agent designed to build and manage WordPress websites through natural-language commands. Angie can generate content, design layouts, write code, configure plugins, and automate routine admin tasks — all within your existing WordPress dashboard.

---

## 1) Getting Started

### Install and Access
- Install “Angie” plugin from WordPress Plugins screen  
- Log into Elementor account when prompted  
- Open from WP Admin toolbar → “Toggle Angie”

### Where Angie Works
- WP Admin dashboard
- Elementor editor
- Block editor (Gutenberg)
- WooCommerce admin pages

> Context-aware: Actions apply to the screen or element you are currently viewing or have selected.

### Permissions
- **Only Administrators** can use Angie

---

## 2) Core Capabilities

### A) Content Creation & Copywriting
- Write and rewrite copy
- Generate full page content from brief descriptions
- Translate content
- Fix SEO and accessibility (alt text, missing meta)

### B) Design & Layout (Elementor)
- Create pages and sections
- Add or replace media (including AI-generated images)
- Modify styling, spacing, animations
- Diagnose layout issues

### C) Development & Code
- Generate HTML, CSS, JavaScript
- Add Custom Code (site-wide CSS or scripts)
- Fix errors and explain code

### D) WordPress Site Management
- Change site settings
- Create and manage users
- Update homepage and reading settings

### E) Plugins & Themes
- Install, activate, update plugins and themes
- Delete unwanted extensions
- Confirmations required for destructive actions

### F) WooCommerce (Store Management)
- Add products and images
- Bulk price updates
- Configure shipping rules
- Sync stock

### G) Third-Party Plugin Integrations
Supports:
- Elementor Pro
- ACF (custom fields)
- JetEngine (CCT, listings)
- The Events Calendar
- GiveWP
- LearnDash
- WS Form

---

## 3) Prompting Best Practices
- Be specific
- Work **in the correct context**
- Select the target element (Elementor)
- Build in steps
- Ask “What can you do?”

---

## 4) Control & Safety
- Preview before applying major changes
- Confirmation for irreversible actions
- Full **Undo History** supported

---

## 5) Known Limitations
- Beta maturity
- Requires Containers (not old Elementor Sections)
- Plugin support expanding but not universal
- Daily AI credit limits on free tier
- Admin-only access

---

## Quick Prompt Examples
- “Audit my site for improvements”
- “Install ACF and create Recipe CPT with fields Ingredients and Instructions”
- “Add a Summer Sale coupon for 10% off all products”
- “Optimize this page layout for mobile”
